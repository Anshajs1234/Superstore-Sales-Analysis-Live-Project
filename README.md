# Superstore Sales Performance and Customer Analysis Project 🚀

**Live Demo:** https://anshaj-superstore-dashboard-live.streamlit.app

## 📊 Project Overview
Interactive sales dashboard built to analyze Superstore dataset with real-time filtering and dynamic visualizations.

## ✨ Key Features
- **Interactive Filters:** Filter data by Year, Region, and Customer Segment
- **KPI Cards:** Total Sales, Profit, Orders with real-time updates  
- **Dynamic Charts:** Sales vs Profit scatter plot, Monthly sales trends
- **Responsive Design:** Works on desktop and mobile

## 🛠️ Tech Stack
- **Python** - Core programming language
- **Streamlit** - Web app framework for dashboard
- **Pandas** - Data manipulation and analysis
- **Plotly** - Interactive charts and visualizations

## 📈 Dataset
Superstore Sales dataset with 9,994 records containing Order ID, Sales, Profit, Region, Category, etc.
